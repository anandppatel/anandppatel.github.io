Lemma 2.9. (01C4)

There exists a filtration $$E = F_{0} \supset F_{1} \supset \dots \supset F_{r-1} \supset F_{r} = 0,$$ satisfying the following properties.

  1. For every $i \in \{0, \dots, r-1\}$, the sub-quotient $F_{i}/F_{i+1}$ is a line bundle.
  2. Set $L_i = F_{i}/F_{i+1}$ for $i \in \{1, \dots, r-1\}$ and $L_r = F_0/F_1$. For every $i \in \{1, \dots, r-1\}$, we have $$\deg L_i + N \leq \deg L_{i+1}.$$
