Lemma 6.5. (0A59)

Suppose $\mu^{\max}(i_0)=\frac{k-1}{k}$ for some $i_0$. Then $\mu^{\max}(n) = \mu^{\max}(n-i_0) + 1$ all $n\geq i_0$. If in addition $B_k^{\up}(i)=B_k^{\low}(i)$ for each $i\leq 3i_0$, then $B_k^{\low}(n)=B_k(n)=B_k^{\up}(n)$ for all $n$.
