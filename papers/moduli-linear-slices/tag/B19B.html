Definition 5.3. (B19B)

Let a sequence $(d_1,r_1),(d_2,r_2),\ldots, (d_a,r_a)$ in $\mathbb{Z}_{\geq 0}\times \mathbb{Z}_{>0}$ be $k$-admissible if $d_1\leq 0$ and $0\leq \frac{d_{i+1}}{r_{i+1}}-\frac{d_i}{r_i}\leq \frac{1}{k}$ for each $i$. Let $A_{k,n}$ denote the set of $k$-admissible sequences with $\sum_{i} r_i=n$ (where $a$ is arbitrary).
