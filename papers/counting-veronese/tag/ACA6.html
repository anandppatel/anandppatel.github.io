Proof. (ACA6)

We will calculate the space of first order deformations of $(T, T^{*}) \in \CT$, and demonstrate that it is $6$ dimensional. This is enough by Proposition 3.14.

Let us pass to affine coordinates; we let $x = X/Z, y = Y/Z, a = \check{X}/\check{Y}, c = \check{Z}/\check{Y}$. The general first order deformation of the ideal $I = (x^{2},xy,y^{2})$ is given by

$$\begin{aligned}I_{\varepsilon} := (x^{2} + \varepsilon(\alpha_{1}x + \beta_{1}y), xy + \varepsilon(\alpha_{2}x + \beta_{2}y), y^{2}+ \varepsilon(\alpha_{3}x+\beta_{3}y)),\end{aligned}$$ for free choices of constants $\alpha_{i}, \beta_{j} \in \k$, while the general first order deformation of $J = (a^{3}, c)$ is $$\begin{aligned}J_{\varepsilon} = (a^{3} + \varepsilon(\gamma_{1}a^{2}+\gamma_{2}a+\gamma_{3}), c + \varepsilon(\delta_{1}a^{2}+\delta_{2}a + \delta_{3}))\end{aligned}$$ where $\gamma_{i}$ and $\delta_{i}$ vary freely in $\k$.

From (?), the corresponding first order deformation of the induced net of conics $\Lambda_{T} = \langle X^{2}, XY, Y^{2} \rangle$ is given by $$\begin{aligned}\Lambda_{T}(\varepsilon) := \langle X^{2} + \varepsilon(\alpha_{1}XZ + \beta_{1}YZ), XY + \varepsilon(\alpha_{2}XZ + \beta_{2}YZ), Y^{2}+ \varepsilon(\alpha_{3}XZ+\beta_{3}YZ) \rangle.\end{aligned}$$

Next, we must identify the first order deformation of the net of conics $\Lambda_{T^{*}} = \langle \check{X}\check{Z}, \check{Y}\check{Z}, \check{Z}^{2} \rangle$ induced by the deformation (?). First, note that $c^{2} \in J_{\varepsilon}$. Therefore, we must determine how the conics $\check{X}\check{Z}$ and $\check{Y}\check{Z}$ must be deformed. By homogenizing the element $c + \varepsilon(\delta_{1}a^{2}+\delta_{2}a + \delta_{3}) \in J_{\varepsilon}$, we get the deformation: $$\begin{aligned}\check{Y}\check{Z} + \varepsilon(\delta_{1}\check{X}^{2}+\delta_{2}\check{X}\check{Y} + \delta_{3}\check{Y}^{2}).\end{aligned}$$ Finally, by multiplying $c + \varepsilon(\delta_{1}a^{2}+\delta_{2}a + \delta_{3})$ by $a$, subtracting $\varepsilon \delta_{1} a^{3}$, then homogenizing we deduce the following deformation: $$\begin{aligned}\check{X}\check{Z} + \varepsilon(\delta_{2}\check{X}^{2} + \delta_{3}\check{X}\check{Y})\end{aligned}$$ Altogether, the corresponding first order deformation of the net $\Lambda_{T^{*}}$ is given by $$\begin{aligned}\Lambda_{T^{*}}(\varepsilon) := \langle \check{X}\check{Z} + \varepsilon(\delta_{2}\check{X}^{2} + \delta_{3}\check{X}\check{Y}), \check{Y}\check{Z} + \varepsilon(\delta_{1}\check{X}^{2}+\delta_{2}\check{X}\check{Y} + \delta_{3}\check{Y}^{2}), \check{Z}^{2} \rangle\end{aligned}$$

Now, we must impose the condition that the pair of deformations $(I_{\varepsilon}, J_{\varepsilon})$ belong in $\CT$. By definition, this is the condition $$\begin{aligned}\Lambda_{T}(\varepsilon)^{\dagger} = \Lambda_{T^{*}}\end{aligned}$$

The left hand side of (?) is simple to calculate. First, the deformation $\Lambda_{T}(\varepsilon)^{*}$ is given by:

$$\begin{aligned}\Lambda_{T}(\varepsilon)^{*} = \langle \check{X}\check{Z} + \varepsilon(-\frac{\alpha_{1}}{2} \check{X}^{2} - \alpha_{2}\check{X}\check{Y} - \frac{\alpha_{3}}{2} \check{Y}^{2}), \check{Y}\check{Z} + \varepsilon(-\frac{\beta_{1}}{2} \check{X}^{2}-\beta_{2}\check{X}\check{Y} - \frac{\beta_{3}}{2} \check{Y}^{2}), \check{Z}^{2} \rangle\end{aligned}$$ and so applying the Jacobian we get

$$\begin{aligned}\Lambda_{T}(\varepsilon)^{\dagger} = \langle \check{X}\check{Z} + \varepsilon(\beta_{2}\check{X}^{2} + (\alpha_{2}-\beta_{3})\check{X}\check{Y} + \alpha_{3}\check{Y}^{2}), \check{Y}\check{Z} + \varepsilon(\beta_{1}\check{X}^{2}+(\beta_{2}-\alpha_{1})\check{X}\check{Y} - \alpha_{2}\check{Y}^{2}), \check{Z}^{2} \rangle\end{aligned}$$ Equating term by term with (?) we get the following system of equations:

$$\begin{aligned}\delta_{2} =\beta_{2}\\ \delta_{3} = \alpha_{2}-\beta_{3}\\ 0 = \alpha_{3}\\ \delta_{1} = \beta_{1}\\ \delta_{2} = \beta_{2}-\alpha_{1}\\ \delta_{3} = -\alpha_{2}\\\end{aligned}$$

We note that this system of equations clearly cuts out a $6$-dimensional space of solutions in the vector space $\k\langle \alpha_{1}, \beta_{1}, \alpha_{2}, \beta_{2}, \alpha_{3}, \beta_{3}, \gamma_{1}, \gamma_{2}, \gamma_{3}, \delta_{1}, \delta_{2}, \delta_{3} \rangle$ – indeed, we may freely choose $\gamma_{1}, \gamma_{2}, \gamma_{3}, \delta_{1}, \delta_{2}, \delta_{3} $, after which the $\alpha$ and $\beta$ variables are uniquely determined, concluding the proof.
