Definition 5.1. (47C1)

Define $c_{i}(j) \in \CH^{i}\CT$ to be the Chern class $c_{i}\left(\O(j)^{[3]}\right)$. Define $e_k \in \CH^{k}\CT$ to be the Chern class $c_{k}(E)$, where $E$ is the rank $9$ vector bundle from Section 3.6. Set $$\begin{aligned}\Delta_{0}:= e_{3}^2-e_{2}e_{4},\\ \Delta_{2}:= e_{2}^{2}-e_{1}e_{3}\\ \Delta_{4}:= e_{1}^{2}-e_{2},\\ \Delta_{6} := [\CT].\end{aligned}$$
