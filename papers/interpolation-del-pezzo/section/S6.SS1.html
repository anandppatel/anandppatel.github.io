6.1. Results

The main result of this section is:

Theorem 6.1. (4EC0)

Let $\Gamma \subset \bp^{9}$ be thirteen general points. Then there exists a $3$-Veronese surface containing $\Gamma$.

Proof assuming Theorem 6.8 [D96B] and Theorem 6.10 [395E]. (69E6)

By Theorem 6.8 [D96B], for a general $\Gamma_{13} \in \Hilb_{13} \bp^2$, there is a bijection between singular triads for $\Gamma_{13}$ (defined below in Definition 6.2 [C70E]) and $3$-Veronese surfaces containing the associated set $A(\Gamma_{13}) \subset \bp^{9}$. By Theorem 6.10 [395E], every such $\Gamma_{13}$ indeed possesses a singular triad.

The essential tool used in proving Theorem 6.1 [4EC0] is association.

Our next result relates the number of Veronese surfaces through $13$ general points to another, more tractable enumerative problem. Before stating it, we must make a definition.

Definition 6.2. (C70E)

Let $\Gamma \subset \bp^2$ be a general set of thirteen points in the plane. A subset $T = \{x,y,z\} \subset \bp^2 \setminus \Gamma$ of three distinct points is a singular triad for $\Gamma$ if $$h^{0}(\bp^{2}, \so_{\bp^{2}}(5)\otimes \sci_{T}^{2}\sci_{\Gamma}) = 2.$$

Remark 6.3. (3A86)

In other words, $T = \{x,y,z\}$ is a singular triad for $\Gamma$ if there exists a pencil of quintic curves through $\Gamma$ and singular at $x, y,$ and $z$. A dimension count shows that we expect finitely many singular triads for a general set of thirteen points $\Gamma$, as is done in Lemma 6.13 [1DB6].

Our second result is:

Theorem 6.4. (E409)

The number of $3$-Veronese surfaces through a general set of thirteen points in $\bp^{9}$ is equal to the number of singular triads for a general set of thirteen points in $\bp^{2}$.

Theorem 6.4 [E409] points to an interesting enumerative problem on the Hilbert scheme ${\rm Hilb}_{3}(\bp^{2})$ of degree $3$, zero dimensional subschemes of the plane. We discuss this problem at the end of the section, in Section 6.7.
