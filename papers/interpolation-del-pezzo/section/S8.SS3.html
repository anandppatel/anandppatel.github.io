8.3. Tools for Showing Equality of Dimensions of the Source and Target

In this subsection we develop some more technical tools for proving  Theorem 8.7 [2AF2]. Our goal for this subsection is to prove  Lemma 8.13 [1A35]. Before embarking on this task, we start with a simple tool for proving the equivalence of (3) and (5).

Lemma 8.12. (2425)

Let $\pi: X \ra Y$ be a proper morphism of locally Noetherian schemes of the same pure dimension. If there is some point $x \in X$ which is isolated in its fiber, then $\dim \im \pi = \dim Y$.

Proof. (40C1)

By Zariski's Main Theorem in Grothendieck's form [Vak, Theorem 29.6.1(a)] there is a nonempty open subscheme $X_0 \subset X$ so that all closed point of $X_0$ are isolated in their fibers. Therefore, the map restricted to this open subset is generically finite, and so its image has the same dimension as $Y$.

Lemma 8.13. (1A35)

With notation as in  Definition 8.3 [2885], if $\sum_{i=1}^m \lambda_i = \dim \hilb X$, we have $\dim \Phi = \dim \prod_{i=1}^m Gr(\codim X - \lambda_i +1, n+1)$. In particular, the source and target of the map $\pi_2$ have the same dimension.

Proof. (D1A2)

This is purely a dimension counting argument. With notation as in Definition 8.3 [2885], $\Phi$ is a fiber product of incidence correspondences, $\Phi_i$, where $\Phi_i$ is the scheme theoretic image of $$\begin{aligned}\Psi_i := \uhilb {\Lambda_i} \times_{\mathbb{P}^n} \scv(U)\end{aligned}$$ under projection map $\Psi_i \ra \hilb X \times Gr(\codim X - \lambda_i + 1, n+1).$ We have natural projections \nonumber

with $$\begin{aligned}\Phi = \Phi_1 \times_{\hilb X} \Phi_2 \times_{\hilb X} \cdots \times_{\hilb X} \Phi_m.\end{aligned}$$ The dimension of any fiber of $\pi_1^i$ is $\dim X + \dim Gr(\codim X-\lambda_i, n)$, and hence $$\begin{aligned}\dim \Phi &= \dim \hilb X + \sum_{i=1}^m (\dim X + \dim Gr(\codim X-\lambda_i, n)) \\ &= \dim \hilb X + \sum_{i=1}^m (\dim X + (\dim Gr(\codim X-\lambda_i + 1, n + 1) - n-\codim X - \lambda_i)) \\ &= \dim \hilb X - \sum_{i=1}^m \lambda_i + \sum_{i=1}^m \dim Gr(\codim X-\lambda_i + 1, n + 1) \\ &= \sum_{i=1}^m \dim Gr(\codim X-\lambda_i + 1, n + 1).\end{aligned}$$
