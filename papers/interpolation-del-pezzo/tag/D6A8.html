Definition 8.5. (D6A8)

Let $\lambda$ be admissible and let $E$ be a locally free sheaf on a scheme $X$ with $H^1(X, E) = 0$. Choose points $p_1, \ldots, p_m$ on $X$ and vector subspaces $V_i \subset E|_{p_i}$ for $1 \leq i \leq m$ with $\codim V_i = \lambda_i$. Then, define $E'$ so that we have an exact sequence of coherent sheaves on $X$

We say $E$ satisfies $\lambda$-interpolation if there exist points $p_1, \ldots, p_n$ as above and subspaces $V_i \subset E|_{p_i}$ as above so that $$\begin{aligned}h^0(E) - h^0(E') = \sum_{i=1}^m \lambda_i.\end{aligned}$$

Write $h^0(E) = q \cdot \rk E + r$ with $0 \leq r < \rk E$. We say $E$ satisfies

  1. weak interpolation if it satisfies $\left( (\rk E)^q \right)$ interpolation.
  2. interpolation if it satisfies $\left( (\rk E)^q, r \right)$ interpolation.
  3. strong interpolation if it satisfies $\lambda$-interpolation for all admissible $\lambda$.
