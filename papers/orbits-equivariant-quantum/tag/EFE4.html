Definition 15.2. (EFE4)

[{[Li09, Definition 2.2]}]

Let $\mc{S}$ be a simple arrangement of subvarieties of $X$. A subset $\mc{G}\subset \mc{S}$ is called a building set of $\mc{S}$ if for all $S\in \mc{S}$, the minimal elements of $\mc{G}$ containing $S$ intersect transversely and their intersection is $S$.

We say that a set $\mc{G}$ of subvarieties of $X$ is a building set if it is a building set of some simple arrangement $\mc{S}$ of subvarieties of $X$. In this case $\mc{S}$ is the set of all proper nonempty intersections of elements of $\mc{G}$.
