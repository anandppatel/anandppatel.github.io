Definition 15.3. (4504)

[{[Li09, Definition 2.12]}]

Let $X$ be a smooth variety and let $\mc{G}$ be a building set of subvarieties of $X$. Order $\mc{G}=\{G_1,\ldots,G_n\}$ so that $G_i\subset G_j$ implies $i < j$, i.e. $G_i$ is a minimal element of $\{G_i,\ldots,G_n\}$ for all $i$. Let $X_0 = X$ and for each $k$ with $0 < k \leq n$, let $X_{k}$ be the blowup of $X_{k-1}$ along $\wt{G_k}$ (the iterated strict transform of $G_k$ in $X_{k-1}$). Define the wonderful compactification $X_{\mc{G}}$ to be $X_n$, which by Proposition 15.5 [F68F] below is independent of the ordering of $\mc{G}$.

By [Li09, Proposition 2.8], all the blowups carried out in Definition 15.3 [4504] have smooth centers, so $X_{\mc{G}}$ is smooth.
