14.3. Resolution of maps

Proposition WCE allows one to resolve a collection of rational maps with smooth base loci via iterated blowups. \begin{Prop} \label{RMBU} Let $X\dashrightarrow X_i$ be a finite set of rational maps with simple base loci $B_i$. If $\mc{G}$ is a building set of subvarieties of $X$ that contains each $B_i$, then the wonderful compactification $X_{\mc{G}}\to X$ resolves the map $X\dashrightarrow \prod X_i$. \end{Prop}
