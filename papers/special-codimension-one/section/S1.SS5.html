1.5. The Maroni loci $\sfM(\cE)$.

The Hurwitz space $\H_{d,g}$ contains natural subvarieties (or substacks) which we call the Maroni loci $\sfM(\cE)$. More precisely, we make the following definition:

Definition 1.7. (8499)

Let $\cE$ be a rank $d-1$, degree $g+d-1$ vector bundle on $\P^1$. Then the Maroni locus $\sfM(\cE)$ is defined as $$\sfM(\cE) := \overline {\{[\alpha \from C \to \P^1] \in \H_{d,g} \mid \cE_{\alpha} = \cE\}}.$$

Clearly if $\sfM(\cE)$ is nonempty then $\deg \cE = g+d-1$ and $\rk \cE = d-1$. However, more can be said about $\cE$ in this situation. For instance, $\cE$ cannot have any nonpositive summands, since this would force $C$ to be disconnected. Write $$\mathcal{E} = \cO(a_{1}) \oplus \cO(a_{2}) ... \oplus \cO(a_{d-1})$$ with $0 < a_{1} \leq a_{2} \leq ... \leq a_{d-1}$. and set $\lfloor \cE \rfloor : = a_{1}$.

Definition 1.8. (D3D7)

We maintain the notation above. A vector bundle $\cE$ of rank $d-1$ and degree $g+d-1$ on $\P^1$ is tame if $a_{i} - a_{i-1} \leq \lfloor \cE \rfloor$ for all $i = 2, ... ,d-1$.

Among all tame bundles $\cE$ with fixed $\lfloor \cE \rfloor = m$, there is a most generic one, which we call $\cE[m]$. It is the unique tame bundle which maximizes the sum $(d-1)a_{1} + (d-2)a_{2} + ... + a_{d-1}$.

The most comprehensive theorem on the geometry of the Maroni loci to date is found in [DP15], which relies heavily on the work of Ohbuchi [Ohb97] and Coppens [Cop99]:

Theorem 1.9. (C76B)

Let $m$ be an integer satisfying $\frac{g+d-1}{{d \choose 2}} \leq m \leq \frac{g+d-1}{d-1} $.

  1. If $\sfM(\cE)$ is nonempty, then $\cE$ is a tame bundle.
  2. If $\lfloor \cE \rfloor \leq m$ then $\sfM(\cE) \subset \sfM(\cE[m])$.
  3. $\sfM(\cE[m]) \subset \sfM(\cE[m+1])$ for all $m$.
  4. $\sfM(\cE[m])$ is an irreducible subvariety of $\H_{d,g}$ of codimension $g-(d-1)m + 1$ unless $m = \lfloor \frac{g+d-1}{d-1} \rfloor$, in which case $\sfM(\cE[m]) = \H_{d,g}$.

In particular, point $(4)$ says that a general cover $\alpha \from C \to \P^1$ has a balanced Tschirnhausen bundle $\cE_{\alpha}$.

The Maroni divisor $\sfM \subset \H_{d,g}$.

Item $(4)$ in Theorem 1.9 [C76B] gives a complete description of all divisorial Maroni loci. Setting the codimension $g-(d-1)m +1$ equal to $1$ gives the divisibility relation $$(d-1) \mid g.$$ When this congruence condition is met, the general Tschirnhausen bundle $\cE^{gen}$ is perfectly balanced, i.e. $$\cE^{gen} = \cO(k)^{\oplus d-1}$$ where $k = g/(d-1) +1$. Consider the special bundle $$\cE^{sp} := \cO(k-1) \oplus \cO(k)^{d-3} \oplus \cO(k+1).$$ Then the Maroni locus $\sfM(\cE^{sp})$ is precisely the locus $\sfM$ from Definition 1.1 [76AC].
