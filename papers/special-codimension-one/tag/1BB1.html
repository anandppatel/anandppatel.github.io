Proof. (1BB1)

First we recall that the Grothendieck-Riemann-Roch formula yields: $$\lambda = \chi (\cO_{\cC}) - \chi(\cO_{S})(1-g).$$ Since $\chi(\cO_{\cC}) = \chi(\alpha_{*}\cO_{\cC}) = \chi(\cO_{\P}) + \chi(\dual \cE)$ we get: $$\lambda = \chi(\cO_{\P}) + \chi(\dual \cE) - (1-g)\chi(\cO_{S}) .$$

We calculate $\chi(\dual \cE)$ by applying the Hirzebruch-Reimann-Roch formula to the bundle $\dual \cE$. This says that $$\chi(\dual \cE) = [ch(\dual \cE) \cdot Td(T_{\P})]_{2} = ch_{2}(\dual \cE) + c_1(\cE)(K_{\P}/2) + (d-1)\chi(\cO_{\P}).$$

Let $\sigma \in \Pic_{\Q} \P$ denote the rational section class $[-\omega_{p}/2]$, so that $\sigma^{2} = 0$. Since the degree of $\cE$ is $g+d-1$ when restricted to any $\P^{1} \times \{s\} \subset \P$, we may write $c_{1}(\cE) = (g+d-1)\sigma + mf$, where $f$ is the class of a fiber of $p$.

Squaring gives $$m = \frac{c_{1}^{2}\cE}{b}.$$ The equality $K_{\P}/2 = -\sigma - \chi(\cO_{S})\cdot f$ implies $$c_{1}(\cE) \cdot K_{\P}/2 = -m - (g+d-1) \chi(\cO_{S}).$$

Altogether, we obtain $$\lambda = ch_{2}(\dual \cE) - m + d\chi(\cO_{\P}) - d\chi(\cO_{S}) = ch_{2}(\dual \cE) - \frac{c_{1}^{2}\cE}{b},$$ since $\chi(\cO_{\P}) = \chi(\cO_{S})$. This gives (3.3).

Mumford's relation $12 \lambda = \kappa +\delta$ implies we need only compute $\kappa$. The computation of $\kappa$ uses the Grothendieck-Riemann-Roch formula applied to the line bundle $\omega_{\alpha}^{\otimes 2}$ and the map $\alpha$ as in the proof of 1.14 [2558]: $$ch(\alpha_{*}\omega_{\alpha}^{\otimes 2})Td(T_{\pi}) = \alpha_{*}(ch(\omega_{\alpha}^{\otimes 2})Td(T_{f})).$$

By using $ch(\alpha_{*}\omega_{\alpha}^{\otimes 2}) = ch(\Sym^{2}\cE) - ch(\cF)$ and taking degree $2$ parts of the above equation, we end up with the equality $$ch_{2}(\Sym^2\cE) - ch_{2}(\cE) - ch_{2}(\cF) = c_{1}^{2}(\omega_{\alpha}).$$

Finally, by using $c_{1}(\omega_{f}) = c_{1}(\omega_{\alpha}) + \alpha_{*}c_{1}(\omega_{\pi})$, one checks that $$\kappa = c_{1}^{2}(\omega_{f}) = c_{1}^{2}(\omega_{\alpha}) - \frac{8}{b}c_{1}^{2}(\cE).$$

Putting all of the above together yields 3.3.
