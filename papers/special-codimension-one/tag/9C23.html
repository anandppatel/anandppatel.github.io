Proof. (9C23)

Let $\cA$ be any vector bundle on $P^1_{t} \times P^1_{s}$ such that $\cA_{t}$ has vanishing higher cohomology for all $t \in \P^{1}_{t}$. We can compute the degree of the pushforward $f_{t*}\cA$ using the Grothendieck-Riemann-Roch formula; we get: $$\deg \, (f_{t*}\cA) = c_1\cA\cdot R_s + \frac{c_1^2\cA}{2}-c_2\cA.$$

We apply this formula to $\cA = \cW(-(l+1)R_s)$. For this, we first note that $$\begin{aligned}&&c_1\cW = c_1\cV + aR_s+R_t = \deg \cV\cdot R_s +a R_s +R_t,\\ &&c_2\cW = \deg \cV + 2a,\end{aligned}$$ and so $$\begin{aligned}c_1\cW(-(l+1)R_s) &=& \deg \cV\cdot R_s +a R_s +R_t - (N-1)(l+1)R_s\\ c_2\cW(-(l+1)R_s) &=& \deg \cV + 2a + (N-2)(-(l+1)R_s)c_1 \cW\\ &=& \deg \cV + 2a - (N-2)(l+1).\end{aligned}$$

Plugging this into (4.13) gives: $$\begin{aligned}\deg \, f_{t*}\cW(-(l+1)R_s) &=& 1 + [\deg \cV + a - (N-1)(l+1)] - [\deg \cV + 2a - (N-2)(l+1)]\\ &=& -a-l.\end{aligned}$$

Now consider the sequence which gives rise to the vector bundle $\cD$: $$0 \longrightarrow f_{t}^*f_{t*}\cW(-(l+1)R_s) \longrightarrow \cW(-(l+1)R_s) \longrightarrow \cD \longrightarrow 0$$ The family of bundles $P\cD_{t}$, $t \in \P^{1}_{t}$, is the family of directrices of $P\cW_t$. If we let $\eta$ be the hyperplane class associated to the $\cO(1)$ on $P\cW(-(l+1)R_s)$, we get the equality $$[P\cD] = \eta^{N-1-r} +(a+l)R_t\cdot \eta^{N-2-r}$$ in the Chow ring of $P\cW(-(l+1)R_s)$. If we denote by $\zeta$ the natural hyperplane class for $P\cW$, we see that $\eta = \zeta - (l+1)R_s$, and so in terms of $\zeta$, $$[P\cD] = \left( \zeta - (l+1)R_s\right)^{N-1-r} + (a+l)R_t\cdot ( \zeta - (l+1)R_s)^{N-2-r}.$$ In the Chow ring of the ambient space $P\cV$,$$[P\cD] = (\zeta + aR_s+R_t)\cdot \left( \zeta - (l+1)R_s\right)^{N-1-r} + (a+l)R_t\cdot ( \zeta - (l+1)R_s)^{N-2-r}$$

Now, we want to understand $P\cD \cap R_s$. In particular, we want to know its class in the Chow ring of $X = P^{N-1}_{\{s=0\}} \times P^1_t$. So we intersect with $R_{s}$ to obtain $$\begin{aligned}[P\cD] \cdot R_s = [\left( \zeta - (l+1)R_s\right)^{N-1-r} + (a+l)R_t\cdot ( \zeta - (l+1)R_s)^{N-2-r}]\cdot [\zeta + aR_s+R_t]\cdot R_s\end{aligned}$$

Using the fact that $R_s^2 = R_t^2=0$, we can easily calculate this product. If we let $H$ and $F$ be the pullbacks of the hyperplane classes in $P^{N-1}_{\{s=0\}} \times P^1_{t}$, then the class of the scroll swept out by the directrices of $\P \cW$ is $$ [P\cD \cap \{s=0\}] = H^{N-r} + (a+l+1)H^{N-r-1}\cdot F,$$ which is what the proposition claims.
