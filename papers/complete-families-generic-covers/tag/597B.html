Lemma 3.10. (597B)
  1. If $d \geq 3$ then $$|A_{d}(x)| = \frac{1}{2}|M_{d}|.$$
  2. If $d=2$ then $|A_{2}(x)| = |M_{2}(x)|$.
  3. If $d=1$ then $|A_{1}(x)| = 0 = |M_{1}(x)|$.
