Lemma 2.7. (4BE9)
The function $y:S\to\mathbb{Z}$ defined by $y(m)=f(m)+r$ satisfies $$y(m_{1})+y(m_{2})+y(m_{3})=0$$ for any valid triple $(m_{1},m_{2},m_{3})\in T$.
