Remark 3.4. (704B)

We claim that $\overline{BN}^1_3$ is nonempty, i.e. $\pi^{-1}BN^1_3$ consists of the main component $\wt{BN}^1_3$ as well as some boundary divisors. For example, take a pencil of plane cubics and mark a base point as $p_8$. Attach it to $\bbP^1$ at another base point and mark seven general points in $\bbP^1$ as $p_1,\ldots, p_7$. We obtain a curve $C$ moving in $\delta_{0 ; \{1,\ldots, 7 \}}$ with the following intersection numbers: $$ C\cdot \lambda = 1, $$ $$ C\cdot \delta_{0; \{1, \ldots, 7\}} = -1, $$ $$ C\cdot \delta_{0; S} = 0, \quad S\neq \{1, \ldots, 7\}. $$ It follows that $C\cdot \pi^{*} BN^1_3 < 0$, hence $\pi^{*}BN^1_3$ contains $\Delta_{0; S}$ for any $|S| = 7$. It would be interesting to calculate directly the class of the main component $\wt{BN}^1_3$.
