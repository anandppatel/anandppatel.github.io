Remark 4.2. (9DB9)

If one could show that the space $Z_6$ consisting of pairs $(p,\ell)$ where $\ell$ meets $X$ to order 6 at $p$ is of expected dimension 3, then the argument in the proof of Theorem 4.1 [0122] would imply the number of 2-planes in $X$ is at most $$\begin{aligned}1800 d - 1370 d^2 + 225 d^3,\end{aligned}$$ which one can compare with the $15d^3$ 2-planes in $X$ in the case $X$ is a Fermat hypersurface. \begin{comment} Mathematica code: PolynomialQuotient[ PolynomialRemainder[ PolynomialRemainder[ H1 (d*H1) ((d - 2) H1 + s1) ((d - 4) H1 + 2*s1) ((d - 6) H1 + 3*s1) ((d - 8) H1 + 4*s1) ((d - 10) H1 + 5*s1), H1^6, H1], H1^2 - s1*H1 + s11, H1], H1, H1] /. {s1^6 -> 5, s1^4 *s11 -> 2, s1^2 *s11^2 -> 1, s11^3 -> 1}

Output: 1800 d - 1370 d^2 + 225 d^3

For the flecnode locus, we recover the usual answer PolynomialQuotient[ PolynomialRemainder[ PolynomialRemainder[ H1 (d*H1) ((d - 2) H1 + s1) ((d - 4) H1 + 2*s1) ((d - 6) H1 + 3*s1), H1^4, H1], H1^2 - s1*H1 + s11, H1], H1, H1] /. {s1^4 -> 2, s1^2 *s11 -> 1, s11^2 -> 1}

Output: -24 d + 11 d^2 \end{comment}
